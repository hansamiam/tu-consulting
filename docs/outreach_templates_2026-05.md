## Send order + cadence

**Order (highest expected ROI first):**

1. **Telegram / IG channel admins** (Archetype 3) — fastest feedback loop, lowest friction, single yes/no decision, audience already warm to study-abroad content.
2. **Test prep centers** (Archetype 5) — clearest mutual interest, owners already think in referral economics.
3. **Independent counselors** (Archetype 2) — slower close but high LTV per partnership.
4. **Parent group admins** (Archetype 4) — webinar takes prep time; do once channel partnerships are validating the funnel.
5. **School principals** (Archetype 1) — slowest sales cycle (committee, term timing); seed now, expect 2–3 month close.

**Cadence per contact:**
- D+0 — initial send (use SHORT for DM channels, MEDIUM for email)
- D+3 — one-line nudge: "Hi [NAME], floating this back up — worth a quick reply if it's a no, so I stop pinging."
- D+10 — final: "Last note from me — closing partnership outreach for May. If it lands later, my line stays open: [CALENDLY]."
- After D+10, drop. Re-approach in 3 months only with a new hook (new feature, new outcome story).

**Highest-conversion bet:** Archetype 3 (channel admins). Reason: it's a transactional ask wrapped as a value gift to their audience. They lose nothing, gain content + goodwill, and the $19/mo code is a real lever they can announce — not a vague "interesting product." Ship these first this week.
